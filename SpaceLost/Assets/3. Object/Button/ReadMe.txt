[버튼/문 사용법]

1. 버튼 프리팹을 설치한다.
2. 자식 오브젝트 중 HitBox를 클릭, 타입을 설정한다.
3. 토글 타입으로 할 경우, Target Tag는 Attack으로 고정해야한다.
4. 프레스 타입일 경우, Target Tag를 비워두면 모든 오브젝트로 상호작용 할 수 있다.
4-1. 특정 오브젝트로만 눌리게 하고싶으면 Target Tag에 상호작용할 오브젝트 태그를 적어주면 된다.
5. Linked Door에 연결시킬 Door 오브젝트를 지정한다.
6. Link된 Door의 인스펙터에도 연결시킨 버튼 오브젝트를 지정한다.
7. DoorOpen과 DoorClose에는 게임 오브젝트면 다 집어넣을 수 있다. 포탈을 지정해서 스위치 상태에 따라 포탈의 위치를 바꾸는것도 가능하다.
